# Angularbook
Mon projet en front-end Angular 6 et Back-end Firebase vous permet de repertorier les livres d'une bibliothèques.
Vous pouvez ajouter une photo de chaque livre qui seront stocker dans la base de donnée proposée par Firebase.
Le back end utilise des API proposé par Firebase permettant de gérer toute la partie connexion inscription et enregistrement necessaire à l'utilisation de l'application.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.3.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
